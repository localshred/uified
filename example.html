Tab Links

The Tab Links UI element is a simple way to define a list of links that simply show/hide the appropriate container elements, creating a fast UI experience to toggle between similarly grouped elements. This is useful in many instances, most notably on a site where the user profile section has multiple sub-sections.

Data View

Users should get the final say on how their data is presented to them, at least, if it doesn't take designers 10 years to customize the different views for them. Two simple ways to display a list of information is in List form (surprise!) and Grid form. If your data is an image and a title, grid form works best (such as iPhoto or Picasa). However, if your data contains variable lengths (such as user profile descriptions), a list view may be a better default fit. The Data View UI allows users to toggle between the two views (without invoking AJAX to rebuild the data).

The Data View UI also supports any other view option you can think of, you just need to define the styles for it.
